LULCODE

Our dream is to create an actively updated superset of LOLCODE with added methods.

We aim to preserve the same cute syntax, but add more features of a mainstream language.

Our main project is creating a LOLCODE to JavaScript transpiler and runner, that opens the door to the wide range of Node Modules out there.
While starting on this project, we realised that lots of things are seeminlgy missing from LOLCODE's v1.2 documentation, and there are no more recent official versions of the documentation.

Once we have a working transpiler that is compatible with LOLCODE v1.2, we'll set about developing our own superset, with the things we think are blatantly missing from LOLCODE.
